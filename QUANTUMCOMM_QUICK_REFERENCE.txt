Quantum Teleportation (5 Steps):

  1. EPR Pair Generation
     ├─ Create Bell state: |Φ+⟩ = (|00⟩ + |11⟩) / √2
     └─ Alice: qubit 1, Bob: qubit 2

  2. Message Encoding
     ├─ Alice encodes message bits into qubit 0
     └─ Apply Hadamard for superposition security

  3. Bell Measurement
     ├─ Alice measures qubits 0 & 1 jointly
     └─ Produces 2 classical bits

  4. Classical Communication
     ├─ Alice sends 2 bits to Bob (public channel)
     └─ No security needed (no information in bits alone)

  5. Unitary Correction
     ├─ Bob applies X gate if c[1]=1
     ├─ Bob applies Z gate if c[0]=1
     └─ Bob's qubit 2 now in original message state

  6. Eavesdropping Check
     ├─ Measure CHSH parameter (S ≤ 2√2 ≈ 2.828)
     └─ If S < 2.4 → ALERT: Eavesdropper detected
