## üõ†Ô∏è Troubleshooting

### Build Fails
```bash
# Check build locally first
npm run build

# Fix any TypeScript/ESLint errors
npm run lint

# Test again
vercel --prod
```

### Permission Denied
- Add `dev@dnalang.dev` to Vercel team members
- Or use Option A (Dashboard connection)

### Domain Not Resolving
- Wait 24-48 hours for DNS propagation
- Use `dig dnalang.dev` to verify DNS records
- Check Vercel domain verification status

### API Routes 404
- Ensure `vercel.json` rewrites are correct
- Check that files exist in `app/api/` directory
- Verify `route.ts` exports are correct

## üîê Security Recommendations

### Headers Configuration

Add to `vercel.json`:

```json
{
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        {
          "key": "X-Content-Type-Options",
          "value": "nosniff"
        },
        {
          "key": "X-Frame-Options",
          "value": "DENY"
        },
        {
          "key": "X-XSS-Protection",
          "value": "1; mode=block"
        },
        {
          "key": "Referrer-Policy",
          "value": "strict-origin-when-cross-origin"
        }
      ]
    }
  ]
}
```

### API Rate Limiting

Implement in API routes:

```typescript
// app/api/middleware/ratelimit.ts
import rateLimit from 'express-rate-limit'

export const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutes
  max: 100 // limit each IP to 100 requests per windowMs
})
```
